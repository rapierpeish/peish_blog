联络

定义

设 M 是一个光滑流形，定义联络映射∇为： ∇ : 𝔛(M) × 𝔛(M) → 𝔛(M)

联络∇必须满足以下三条性质： 1. ℝ双线性： ∇aX + bYZ = a∇XZ + b∇YZ,  ∀a, b ∈ ℝ 2. 第一个变量线性: ∇fXY = f∇XY 3. 第二个变量满足莱布尼茨法则： ∇X(fY) = XfY + f∇XY

其中 : - X 和 Y 是向量场，∇XY 是 Y 沿 X 方向的[[协变导数]]（covariant derivative） - Xf 的意思就是 向量场 X 对函数 f 的作用，也就是f 沿着 X 方向的方向导数 - fX 就是把向量场 X 乘上函数 f： (fX)(p) := f(p)X(p)

联络 (connection)：∇ 是一个对象：它告诉你如何在流形上定义“沿方向的导数”

联络在抽象层是一个双线性映射，利用[[双线性映射]]在抽象层定义了联络，[[协变导数]]是联络的代数实现。 在光滑流形 M 上，对所有点沿着某个方向 x 做微小移动，这个方向就是一个向量场 X。向量场 Y 是向量场 X 在这些方向上的变化，这正是联络的核心作用。

光滑1-形式

设 M 是光滑流形，TpM 是点 p 的切空间。 1-形式α是从从TpM切向量到实数的映射： αp : TpM → R  x ↦ αp(x),  x ∈ TpM

推论1：协变导数是光滑1-形式。

联络 1-形式 (Connection 1-forms)

联络 1-形式定义为： ωji = ⟨∇Xej, ej⟩ - 联络 ∇：一个几何结构，定义了如何做“导数”，是抽象的、全局的。 - 联络 1-形式 ωji：在某组标架下，把联络具体化为一组矩阵值的 1-形式

在带有正交标架 {e1, …, en} 的切空间中，定义 联络 ωji 为： ∇ej = ∑iωji ei - 其中 ∇ej 是向量场 ej 的[[协变导数]]（沿任意方向的导数）。
- 当点 p 微小位移时，切向量的变化是沿着切平面进行的，表明切平面标架 {e1, e2}发生了变化。这种变化是无挠的，即切面标架只会在切平面内发生变化，而不沿法线方向旋转。实际上捕捉的是 基向量的变化率，联络 1-形式是描述平面标架 {e1, e2}如何随着曲面上的点的位移而变化的工具。

联络1-形式和[[曲面的自然标架运动#Gauss公式（切向量场的导数分解）| Guass公式]] 关系

高斯公式： ∇rurv = Γ121ru + Γ122rv + M

ωji的几何意义

标架旋转率

  • ωji(X) 表示标架 ej 沿方向 X 的变化投影在 ei 上的量。
  • 例如二维曲面，e1, e2 正交单位切向量： ω12(X) = −ω21(X) 就是 e1 沿 X 方向绕 n 旋转的瞬时速率。

协变导数的局部实现

对偶1-形式 θi

对偶1-形式是映射，作用在切向量上，输出一个实数： θi : TpM → ℝ  x ↦ θi(x),  x ∈ TpM ## 克罗内克符号δji 定义δji： $$ delta^i_j = \begin{cases} 1 & i = j \\ 0 & i \neq j \end{cases} $$ - 作用：表示单位分量，起“对角标识”作用

公式意义 θi(ej) = δji：θi(ej) 提取出向量ej在切向量分量ei上的成分标量： $$ \theta^i(e_j)= \begin{cases} 1 & i = j \\ 0 & i \neq j \end{cases} $$ ## 小例子

设二维曲面上局部切标架 e1, e2​，任意向量： X = 3e1 + 5e2

对偶 1-形式： θ1(X) = 3  θ2(X) = 5

设 {e1, …, en} 是流形 M 上的局部切标架，对偶 1-形式 {θ1, …, θn} 定义为： θi(ej) = δji

  • θi 是 1-形式（作用在向量上得到标量）
  • 每个 θi 与一个切标架 ei 对应

几何意义：把向量“拆解”到标架方向上的分量，就像坐标投影尺。

对任意向量场 Y = ∑jYjej： ∇XY = ∑jX(Yj)ej + ∑i, jYjωji(X)ei - 第一项：标量变化
- 第二项：标架旋转贡献，由 ωji 给出

Levi-Civita 定理

黎曼流形

一个黎曼流形是一个二元组 (M, g)，其中:

  • M 是光滑流形
  • g 是M上内积的度量，是个对称正定映射。

一般光滑曲面都是里面流形。

定理

对任意黎曼流形 (M, g) ，存在唯一一个联络 ∇ 同时满足无挠性和度量相容性。这就是Levi-Civita 联络。

  • 无挠性：∇XY − ∇YX = [X, Y]
  • 度量相容性: ∇g = 0 ⇒ X(g(Y, Z)) = g(∇XT, Z) + g(T, ∇X, Z)

计算

方法 1：通过协变导数展开

  1. 给定切标架 {ei}
  2. 对每个标架 ej，计算其沿方向 X 的协变导数： ∇Xej
  3. 将 ∇Xej 在标架 {ei} 上展开： ∇Xej = ∑iωji(X)ei
  4. 得到 ωji(X) 即联络 1-形式在方向 X 的值。

例1：计算双曲面的联络

两叶型双曲面的上半叶参数化： r(u, v) = (sinh ucos v, sinh usin v, cosh u),   u > 0, v ∈ (0, 2π). - 关于 u, v 的偏导数 ru = (cosh ucos v, cosh usin v, sinh u),  rv = (−sinh usin v, sinh ucos v, 0).

第一基本形式 E = ⟨ru, ru⟩ = cosh2u + sinh2u = cosh 2u,  F = ⟨ru, rv⟩ = 0,  G = ⟨rv, rv⟩ = sinh2u.

ds2 = E du2 + G dv2 = cosh 2u du2 + sinh2u dv2.

由于 F = 0 且 E = E(u), G = G(u)，可用正交坐标的快捷公式： $$ \begin{aligned} &\Gamma^u_{uu}=\tfrac{1}{2}E^{-1}E_u, \qquad \Gamma^u_{vv}=-\tfrac{1}{2}E^{-1}G_u, \qquad \Gamma^u_{uv}=\Gamma^u_{vu}=0,\\[4pt] &\Gamma^v_{uv}=\Gamma^v_{vu}=\tfrac{1}{2}G^{-1}G_u,\qquad \Gamma^v_{vv}=\tfrac{1}{2}G^{-1}G_v=0,\qquad \Gamma^v_{uu}=-\tfrac{1}{2}G^{-1}E_v=0. \end{aligned} $$

代入 E = cosh 2u,  Eu = 2sinh 2u,   G = sinh2u,  Gu = 2sinh ucosh u

计算Christoffel符号：

得非零项 $$ \boxed{ \Gamma^u_{uu}=\tanh 2u,\quad \Gamma^u_{vv}=-\,\frac{\sinh u\cosh u}{\cosh 2u},\quad \Gamma^v_{uv}=\Gamma^v_{vu}=\coth u } $$

于是协变导数（坐标基）为 $$ \begin{aligned} \nabla_{\partial_u}\partial_u&=\tanh 2u\,\partial_u,\\ \nabla_{\partial_u}\partial_v&=\nabla_{\partial_v}\partial_u=\coth u\,\partial_v,\\ \nabla_{\partial_v}\partial_v&=-\frac{\sinh u\cosh u}{\cosh 2u}\,\partial_u. \end{aligned} $$

取单位正交切标架 $$ e_1=\frac{1}{\sqrt{E}}\partial_u=\frac{1}{\sqrt{\cosh 2u}}\partial_u,\qquad e_2=\frac{1}{\sqrt{G}}\partial_v=\frac{1}{\sinh u}\partial_v, $$ 对应正交对偶 1-形式 $$ \theta^1=\sqrt{E}\,du=\sqrt{\cosh 2u}\,du,\qquad \theta^2=\sqrt{G}\,dv=\sinh u\,dv, $$ 使得 ds2 = (θ1)2 + (θ2)2.
