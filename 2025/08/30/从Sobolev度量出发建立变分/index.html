从Sobolev度量出发建立变分

本文从函数空间的角度出发，构造变分法的理论基础，逐步推导欧拉–拉格朗日方程。我们以路径最短问题为例，在具有度量结构的 Sobolev 空间上分析泛函的极值问题。

泛函与变分的基本概念

泛函的说明

泛函（functional）是个映射，满足： L : ℱ → ℝ 泛函将任意 γ ∈ ℱ 作为输入，输出一个实数，ℱ是一个函数集合。 ### 例子：路径弧长 设路径函数 γ(t) : [0, 1] → ℝn，则弧长泛函为： L[γ] = ∫01∥γ′(t)∥dt ### 什么是变分？ “变分”即研究泛函在函数的微小扰动下的变化率，本质上是研究泛函在函数空间中的“方向导数”。 要研究方向导数，必须先定义距离度量。 ## 路径函数空间与 Sobolev 度量 设路径函数为： γ : [0, 1] → ℝn

定义路径空间为一阶Sobolev空间： H1([0, 1], ℝn) = {γ ∈ L2 | γ′ ∈ L2} 在该空间中定义内积： ⟨γ1, γ2⟩H1 = ∫01γ1(t) ⋅ γ2(t) dt + ∫01γ1′(t) ⋅ γ2′(t) dt 由此诱导范数与度量： ∥γ∥H1 = (∫01∥γ(t)∥2 dt + ∫01∥γ′(t)∥2 dt)1/2 d(γ1, γ2) = ∥γ1 − γ2∥H1

该度量满足： - 非负性； - 分离性（在几乎处处等价类中）； - 对称性； - 三角不等式。 因此 H1 是一个完备的度量空间（希尔伯特空间）。 > 空间 = 集合 + 度量

解释：函数的微小领域（Neighborhood）

设泛函定义在路径空间 ℱ = H1([0, 1], ℝn) 上。
给定某个路径函数 γ ∈ ℱ，使用epsilon-delta方式描述微小领域：

函数γ̃ ∈ ℱ 属于γ的一个微小邻域， 如果对任意 δ > 0，满足 ∥γ̃ − γ∥H1 < δ ，那么满足条件的所有γ̃就成为γ的微小领域。

这个“邻域”是通过 Sobolev 范数构建的，衡量路径本身和导数的差异： ∥γ − γ̃∥H12 = ∫01∥γ(t) − γ̃(t)∥2+∥γ′(t) − γ̃′(t)∥2dt ### 局部 Lipschitz 条件 设泛函L : H1 → ℝ。我们说 L 在 γ 的某邻域中满足局部 Lipschitz 条件，是指存在常数C > 0，使得对所有γ̃满足： ∥γ̃ − γ∥H1 < δ  ⇒  |L[γ̃] − L[γ]| ≤ C∥γ̃ − γ∥H1 这个条件说明：泛函的值变化是收敛的，它是可微性的一个必要前提。

从 Lipschitz 到 Gateaux 可微

泛函可微重要性

在一般的泛函空间中，泛函未必处处可微。例如： - 某些非光滑泛函在不可导点不可微； - 如果泛函在扰动方向不连续，也会使变分导数不存在。 ### 泛函可微性（Gateaux 可微） 设L : H1 → ℝ是定义在Sobolev空间上的泛函。如果对任意γ ∈ H1有Lipschitz条件成立，所有对所有方向η ∈ H1，极限存在： $$ \lim_{\epsilon \to 0} \frac{L[\gamma + \epsilon \eta] - L[\gamma]}{\epsilon} $$ 我们说 L 在γ点沿η方向 Gâteaux 可微，

定义变分：Sobolev 空间中的方向导数

给定泛函： L[γ] = ∫01F(γ(t), γ′(t)) dt 引入扰动函数 η ∈ H01([0, 1], ℝn，满足端点条件 η(0) = η(1) = 0。 构造扰动路径： γϵ(t) = γ(t) + ϵη(t) 定义变分（η方向导数）定义为： $$ \delta L[\gamma](\eta) := \left. \frac{d}{d\epsilon} L[\gamma_\epsilon] \right|_{\epsilon = 0} = \left. \frac{d}{d\epsilon} \int^{1}_{0}L(\gamma_{\epsilon},\gamma_{\epsilon}^{'},t)dt \right|_{\epsilon = 0} $$ 导数符号$\frac{d}{d\epsilon}$推进到积分号内： $$ \delta L[\gamma](\eta) =\int^{1}_{0} (\frac{\partial L}{\partial \gamma_{\epsilon}}\eta + \frac{\partial L}{\partial \gamma_{\epsilon}^{'} } \frac{d \eta}{d \epsilon} )dt \quad \epsilon = 0 $$ > 这里使用的多元函数的1阶微分形式： $$ \frac{dF(x(t),y(t))}{dt} = \frac{\partial F}{\partial x} \frac{dx}{dt} +\frac{\partial F}{\partial y} \frac{dy}{dt} $$ 对第二项做分部积分：

部分积分公式 ∫abuv′dx = [uv]|ab − ∫abu′vdx

$$ \delta L[\gamma](\eta) = \int^{1}_{0} \frac{\partial L}{\partial \gamma_{\epsilon}^{'} } \frac{d \eta}{d \epsilon} dt = \int^{1}_{0} \frac{\partial F}{\partial \gamma_{\epsilon}} \underbrace{\frac{\partial \gamma_{\epsilon}}{\partial \epsilon}}_{\eta} dt - \int^{1}_{0} \frac{d}{dt} (\frac{\partial F}{\partial \gamma_{\epsilon}^{'} })\eta dt $$ 合并化简： $$ \delta L[\gamma](\eta) = \int^{1}_{0} (\frac{\partial F}{\partial \gamma_{\epsilon}} - \frac{d}{dt}\frac{\partial F}{\partial \gamma_{\epsilon}})\eta dt $$

δL[γ](η)取极值点： $$ \int^{1}_{0} (\frac{\partial F}{\partial \gamma_{\epsilon}} - \frac{d}{dt}\frac{\partial F}{\partial \gamma_{\epsilon}})\eta dt = 0 $$ 所以： $$ \frac{\partial F}{\partial \gamma_{\epsilon}} - \frac{d}{dt}\frac{\partial F}{\partial \gamma_{\epsilon}} = 0 \Leftrightarrow \frac{\partial F}{\partial \gamma} - \frac{d}{dt}\frac{\partial F}{\partial \gamma} = 0 $$ 这个就是欧拉-拉格朗日方程
