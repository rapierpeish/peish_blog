02.同调群

回忆单纯形理论中关于边界算子 ∂n 的概念：

边界算子 ∂n 的定义

$$ \partial_n(\sigma) = \sum_{i=0}^n (-1)^i [v_0, \dots, \widehat{v_i}, \dots, v_n] $$

边界算子的像

Im(∂n) := {∂n(σ)|σ ∈ Cn(K)} ⊂ Cn − 1(K)

边界算子的核

ker(∂n) := {σ ∈ Cn(K)|∂n(σ) = 0}

计算单纯复形的边界

给定一个简单三角网格：网格只有三个面 [f0, f1, f2] ，中间有一个孔洞。以此网格演示计算 ker(∂1) 核 Im(∂1) 。因为 ∂2 未定义，所有无法计算 Im(∂2)

计算 ker(∂1) 相当于计算网格中所有链环路，图中有4个链环路：

$$ \begin{pmatrix} \{[v_{0},v_{3}],[v_{3},v_{1}],[v_{1},v_{0}] \} = f_{0}\\ \{[v_{1},v_{4}],[v_{4},v_{2}],[v_{2},v_{1}] \} = f_{1}\\ \{[v_{3},v_{5}],[v_{5},v_{4}],[v_{4},v_{3}] \} = f_{2}\\ \{[v_{3},v_{1}],[v_{1},v_{4}],[v_{4},v_{3}] \} \\ \end{pmatrix} $$

计算 Im(∂2) 相当于计算 n + 1 所有单纯形的边界： $$ \begin{pmatrix} \{[v_{0},v_{3}],[v_{3},v_{1}],[v_{1},v_{0}] \} = f_{0}\\ \{[v_{1},v_{4}],[v_{4},v_{2}],[v_{2},v_{1}] \} = f_{1}\\ \{[v_{3},v_{5}],[v_{5},v_{4}],[v_{4},v_{3}] \} = f_{2}\\ \end{pmatrix} $$

结合图示和计算对 ker(∂1) 和 Im(∂2) 观察分析：

  • ker(∂1) 是 1-单纯形构成的回路集合
  • Im(∂2) 是2-单纯形的边界集合

我们构造商群 H1:

H1 = ker(∂1)/Im(∂2)

构造商群的意义是：将 Im(∂2) 从 ker(∂1) 中剔除掉，然后剩下的就是网格的孔洞了。H1 的正式名称是单纯复形 K (三角网格) 的 H1 同调群。接下来我将对同调群做正式完整的介绍。

在定义同调群之前我们必须先证明这个问题：

Im(∂k + 1) ⊆ ker(∂k)

如果这个等式不成立，那么商群的构造就没有意义。接下来将使用链复形这个概念证明等式成立。

链复形

先思考第一个约束条件：

∂k ⋅ ∂k + 1 = 0

这个约束条件说的是：边界的边界是空的，也就是说边界没有边界，两次映射是个零同态。 给定一个三角形 σ = [v0, v1, v2]，很容易就计算出 ∂1 ⋅ ∂2 = 0

从Im(∂k) 定义出发，如果 x ∈ Im(∂k)，那么存在 τ ∈ Ck + 1(K) 使得 x = ∂k + 1(τ)

将 x = ∂k + 1(τ) 带入约束条件： ∂k ⋅ ∂k + 1(τ) = ∂k(x) = 0

从这里可以得到结论：对任意 x ∈ Im(∂k + 1)，x 同时属于 ker(∂k)， 即集合关系成立： Im(∂k + 1) ⊆ ker(∂k)

给定单纯复形 K 情况下，我们可以将同态映射关系推广，构造出同态映射序列，即链复形，其定义如下：

$$ \dots \xrightarrow{\partial_{n+2}} C_{n+1}(K) \xrightarrow{\partial_{n+1}} C_n(K) \xrightarrow{\partial_{n}} C_{n-1}(K)\longrightarrow \dots $$

同调群

综合上面的叙述，我们得到第1个结论：Im(∂k + 1) 是 ker(∂k) 的子群。但是商群的定义依赖于[[正规子群]]，我们还需要证明其正规子群性质。

正规子群定义： 设 G 是一个群，H 是 G 的一个子群。如果对于所有的 g ∈ G 和 h ∈ H，都有 ghg−1 ∈ H，则称 H 是 G 的正规子群，记作 H ⊲ G。

证明Im(∂k + 1) 是 ker(∂k) 的正规子群 令 g ∈ ker(∂k), h ∈ Im(∂k + 1)，其上的二元运算为加法，则： ghg−1 = g + h + (−g) = h ∈ H > g−1 代表逆元

同调群的正式定义： 给定单纯复形 K ,其k-同调群 Hk定义为： Hk = ker(∂k)/Im(∂k + 1)

同调群的拓扑不变性

拓扑不变性

在拓扑学中，[[同胚]]的拓扑空间必然拥有相同的属性 ，这种属性被称其为 “拓扑不变性”（Topological Invariance）。

对同胚的解释： 同胚是拓扑空间之间的 “强等价关系”。同胚是两个拓扑空间X 和 Y 之间的一个双射映射([[双射]]) f : X → Y，并且满足:

  • f 是连续的
  • 逆映射也是连续的

同胚直观理解：曲面可以通过连续变形变为另一个形状，而不发生撕裂和穿插。比如球可以通过连续变形为椭圆。

相同的属性的理解：无论空间的具体形态如何，只要拓扑结构等价（同胚），拓扑不变性就不会改变。球面和椭圆为同胚关系，两个图形的三角剖分的欧拉示性数相同，则欧拉示性数是拓扑不变量。

常见拓扑不变量：

  • 基础属性：连通分支数；紧致性
  • 代数不变量：基本群πk，同调群Hk

同调群拓扑不变性的解释

若拓扑空间 X 与 Y 同胚，则对任意正式 k ≥ 0，其 k-维同调群同构，Hk(X) ≅ Hk(Y)。 同调群描述的是空间的拓扑结构，与具体形态无关。H0同调群捕捉了连通分支数；H1捕捉了1 维洞空洞的数量；H2则是捕捉了独立 2 维封闭曲面数量。

工程应用：

  • 3D 模型封闭性检测：若 3D 模型的​ H2 ≅ ℤ(β2 = 1)，说明模型是 “单封闭曲面”（如球体模型），无 “开口” 或 “通孔”，可用于判断模型是否适合 3D 打印。
  • 形状匹配：两个 3D 模型若同胚，其​ H2 的贝蒂数​ β2 必然相等，可通过 β2 快速筛选拓扑匹配的模型。
