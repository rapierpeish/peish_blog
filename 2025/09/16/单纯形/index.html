1.单纯形

单纯形

从仿射组合出发的定义

给定 k + 1 个[[仿射组合#仿射无关]]的点 S = v0​, v1​, …, vk​在仿射空间Rn(k <  = n) ，由这些点生成的k-单纯形σ 是它们的所有凸组合构成的集合：

$$ \sigma = \left \{ \sum^{k}_{i=0}\lambda_{i}v_{i} \mid \sum^{k}_{i=0}\lambda_{i} = 1 , \lambda_{i} \geq 0 \right \} $$

几何意义：单纯形是这些点的凸包（Convex Hull），即包含这些点的最小凸集

常见例子：

  • 0-单纯形：单点。
  • 1-单纯形：线段。
  • 2-单纯形：三角形（包含内部）。
  • 3-单纯形：四面体（三维空间的三角锥）。

维度

一个单纯形 σ 的维度，定义为其最大顶点个数减 1，即： dim (σ) = |v0, v1, …, vn|−1 = n

非退化单纯形

给定一个 n-维单纯形 σ = [v0, v1, …, vn]，它是非退化的是指：当且仅当其非空顶点子集 v0, v1, …, vi [[仿射组合#仿射无关]]。也就是说，v1 − v0, …, vi − v0 是 ℝm 中线性无关的一组向量（其中 σ 被嵌入在某个 m ≥ n 的空间中）。

举例： [v0, v1, v2] 是一个 2-单纯形（三角形），它的非退化面包括： 1. 边（1-单纯形）：[v0, v1], [v1, v2], [v0, v2]； 2. 顶点（0-单纯形）：[v0], [v1], [v2]。

退化单纯形

一个单纯形 σ = [v0, v1, …, vn] 被称为退化的，若其顶点集合不仿射无关，即： - 存在某种线性关系使得顶点处于一个低于 n 维的仿射子空间中 - 或者顶点中存在重复点

非退化面

是指由 σ 的真子集生成的单纯形，且该子集生成的单纯形本身也非退化。 例：v0, v1 是 v0, v1, v2 的非退化面（1D）。

退化面

若由 σ 的子集生成的单纯形是退化的，则称之为退化面。

单纯形（Simplex） │ ├─ 非退化单纯形（顶点仿射无关） │ ├─ 非退化面（维度低，仍仿射无关） │ └─ 退化面（维度低，仿射相关或点重复） │ └─ 退化单纯形（顶点仿射相关或重复） └─ 无真正几何结构，不产生非退化面

单纯复形

设 K 是 ℝn 中一组有限个单形的集合，若 K 满足以下两个核心公理，则称 K 为一个单纯复形：

闭包公理 若 σ ∈ K（即 σ 是 K 中的单形），则 σ 的 子结构 都属于 K

对于子结构的解释：给定一个三角网格，这是一个单纯复形。单纯复形中一定包含面，由于点和边属于面，那么点和边一定也属于单纯复形。

交公理 对任意两个单形 σ, τ ∈ K，它们的交集 σ ∩ τ，要么是空的；要么就存在一个单纯形 η 满足：

  • η ⊂ σ 并且 η ⊂ τ

三角网格是单纯复形。三角网格其核心构成是：基本单元：0 维顶点（Vertex）、1 维边（Edge）、2 维三角形（Triangle）。很明显三角网格满足闭包公理和交公理。

定向性

单纯形的定向性完全由其顶点的线性顺序决定，不同维度的单纯形（0 维、1 维、2 维、3 维等）定向规则一致

对一个 k 维单纯形 σk = ⟨v0, v1, …, vk⟩（尖括号表示顶点的有序排列）

边界

单纯形的边界

 k-维单形是由仿射无关的顶点组构成的，那么任意减去其中一个定点，得到的顶点组依旧是仿射无关的，并且可以因此构成一个 k-1维单形。

0-单纯形是点，没有边界。 1-单形是线段，去掉一点得到，0-单形点 2-单形是三角形，去掉一点得到，1-单形棱 3-单形是四面体，去掉一点得到，2-单形三角形

去掉一点得到k维单形的边界，这种操作方法被称为单形的边界算子 ∂n。

边界算子 ∂n 正式定义： 对一个定向 n-单形 σ = [v0, v1, …, vn]，其边界算子∂n 是一个映射： $$ \partial_n(\sigma) = \sum_{i=0}^n (-1)^i [v_0, \dots, \widehat{v_i}, \dots, v_n] $$

其中 :

  • $\widehat{v_i}$ 表示去掉第 i 个顶点，得到一个 (n − 1)-单形
  • (−1)i 决定了单纯形的定向性。

边界算子的例子

对2-单纯形 σ = [v0, v1, v2] 应用算子 ∂2 ： $$ [v_{0},v_{1},v_{2}] \xrightarrow{\partial_{2}} [v_1,v_{2}] - [v_{0},v_{2}]+[v_{0},v_{1}] $$

对3-单纯形 σ = [v0, v1, v2, v3] 应用算子 ∂3 ：

$$ [v_{0},v_{1},v_{2},v_{3}] \xrightarrow{\partial_{3}} [v_1,v_{2},v_{3}] - [v_{0},v_{2},v_{3}]+[v_{0},v_{1},v_{3}]-[v_{0},v_{1},v_{2}] $$

链群

对一个单纯复形 K，它的 n-链群 定义为：

Cn(K) = {∑iaiσi | ai ∈ ℤ}

链群是所有 n-单纯形的形式化线性组合 很明显集合 Cn(K) 满足群定义，并且具备加法和交换律，所以链群是个[[阿贝尔群]]。但是链群上存在多种 “乘法” 定义，要小心对待。

链群的加法

给定： c1 = ∑aiσi  c2 = ∑biσi

链群的加法定义为： c1 + c2 = ∑(ai + bi)σi

将链群概念和边界算子结合起来可得到结论：

  1. 若 c1, c2 是链群，那么有： ∂(c1 + c2) = ∂(c1) + ∂(c2)

边界算子的代数结构

结合链群的概念，∂n：边界算子，定义在生成元 σ = [v0, …, vn] 上：

$$ \partial_n(\sigma) = \sum_{i=0}^n (-1)^i [v_0, \dots, \widehat{v_i}, \dots, v_n] $$

从集合映射角度来看，边界算子 ∂n 是个映射，满足：

∂n : Cn(K) → Cn − 1(K)

我提出命题1：边界算子 ∂n是个[[群同态]]

证明命题1：边界算子 ∂n 是个群同态

群同态定义：

设 (G, ⋅) 和 (H, *) 是两个群，映射 ϕ : G → H 称为群同态，当满足： ∀a, b ∈ G, ϕ(a ⋅ b) = ϕ(a) * ϕ(b) (G, ⋅) 和 (H, *) 是满足乘法[[二元运算]]的群

从定义出发，(Cn(K), +), (Cn − 1(K), +) 是两个群，同时边界算子映射成立： ∂n(c1 + c2) = ∂n(c1) + ∂n(c2), c1, c2 ∈ Cn(K) 所以∂n 是个群同态。

讨论像与核

我们已经证明∂n是群同态，接下来就能按照代数方法建立边界算子 ∂n 的[像]与[核]这两个概念。

像

像抽象定义：

设 ϕ : G → H 是[[群同态]]，则其像（Image）定义为： Im(ϕ) := {ϕ(g) ∣ g ∈ G} ⊆ H

给定单纯复形 K, ∂n 的像定义为： Im(∂n) := {∂n(σ)|σ ∈ Cn(K)}

Im(∂n) 称为 n-边界群（n-boundaries），表示(n + 1)-链的边界

Im(∂n) 是个满足群公理的集合，具有加法运算。

核

核抽象定义：

设 ϕ : G → H 是[[群同态]]，其核定义为： ker (ϕ) := {g ∈ G ∣ ϕ(g) = eH} 其中 eH 是 H 的单位元。

边界算子 ∂n 的核 ker(∂n)： $$

ker({n}) := {C{n}(K) | _{n}() = 0 } $$

Cn − 1(K) 是个阿贝尔加法群，所以0是群的单位元，因为零元加上任何元素都是本身，符合单元元定义。

将 ker(∂n) 称为 n-循环群（n-cycles），表示没有边界的n-链

内部

单纯形的内部

给定n-单纯形 σn，其内部集合记作 int(σn)，定义为： $$ int(\sigma^n)= \left\{ x = \sum_{i=0}^{n} \lambda_{i}v_{i} | \sum_{i=0}^{n} \lambda_{i}=1 ,\lambda_{i}>0 \right\} $$

  • 对于1-单纯形，int(σ1) 就是挖空线段两点后的开集
  • 对于2-单纯形，int(σ2) 就是三角形内部的开集
  • 对于2-单纯形，int(σ3) 就是四面体包裹的内部集合的开集

与边界的关系： $$ \partial_{n} \sigma^{n} = \sigma^{n}\text{\\}int(\sigma^n) $$

单纯形的内部与多面体的关系

多面体

多面体是单纯复形的几何实现。单纯复形是定义在拓扑空间上的”抽象接口”，多面体基础了单纯复形的抽象定义，在几何空间上以点、线、面的形式实现。

多面体定义 单纯复形K 对应的多面体记作 |K|，是K中所有单纯形的并集，并且|K|中的开集也在ℝn中，数学表达式为： |K| = ⋃σ ∈ Kσ ⊂ ℝn 常见多面体：

  • 0 - 维多面体：单个顶点（对应仅含 0 - 单纯形的复形）或有限个孤立顶点（对应含多个不相交 0 - 单纯形的复形）
  • 1 - 维多面体：线段（单个 1 - 单纯形的并集）、折线（多个 1 - 单纯形共享顶点拼接）、圆的单纯剖分（如正六边形剖分为 6 个 1 - 单纯形，多面体为圆周）​
  • 2 - 维多面体：三角形闭区域（单个 2 - 单纯形的并集）、圆盘的剖分（多个 2 - 单纯形共享边拼接，多面体为圆盘）、球面的剖分（如正八面体剖分为 8 个 2 - 单纯形，多面体为球面）

多面体分离性原理

多面体的分离性是指：空间中任意两个不同点，总能能找到两个不相交的开集分别包含它们。这是拓扑空间的豪斯多夫公理。多面体的分离性本质依赖单纯形内部的互不相交性与子空间拓扑的遗传性，以下以最豪斯多夫性质为例展开说明。

[[豪斯多夫性质|豪斯多夫空间]]的定义（分离性核心标准） 若拓扑空间X满足： 对任意x, y ∈ X 且 x ≠ y，存在 X 中的开集 U, V，使得x ∈ U, y ∈ V 且 U ∩ V = ∅，则称 X 为豪斯多夫空间。 我将提出：多面体 |K| 是典型的豪斯多夫空间。

证明：多面体 |K| 是豪斯多夫空间 设​ K 是单纯复形 K 的多面体，任取​x, y ∈  ∣ K∣ 且​ x ≠ y，需分两种情况构造不相交开集：

情况1： x, y 属于同一个单纯形的内部

  • 设 x, y ∈ int(σ)，因为int(σ) 是开集并且 int(σ) 是 ℝn 的子空间，ℝn 是豪斯多夫空间，其子空间也基础豪斯多夫性质。所以 int(σ) 中存在 U, V 使得 x ∈ U, y ∈ V 且 U ∩ V = ∅

情况2: x, y 不属于同一个单纯形的内部

  • 设 x ∈ int(σ1), y ∈ int(σ2)(σ1 ≠ σ2)，因为单纯形内部是不重叠的，所以 int(σ1) ≠ int(σ2)
  • 对于 x ∈ int(σ1)，可以取开球B(x, ϵ1) 满足 B(x, ϵ1) ∩ int(σ1) ⊂ int(σ1)；对于y 做相同操作，得到开球 B(y, ϵ2) ∩ int(σ2) ⊂ int(σ2)
  • 因为 int(σ1) ∩ int(σ2) = ∅，所以B(x, ϵ1) ∩ B(y, ϵ2) = ∅，满足豪斯多夫分离性公理

多面体分离性的应用 多面体的分离性确保了单纯逼近定理是有效的。 单纯逼近定理：

  • 若f : |K| → |L| 是连续映射，若|K| 满足豪斯多夫分离公理并且是紧致的，可通过重心重分构造单纯映射​ g : K′ → L 逼近f 构造同伦等价。而单纯逼近是证明 “同调群是拓扑不变量” 的关键步骤，最终支撑多面体在流形剖分、有限元分析等领域的应用。
